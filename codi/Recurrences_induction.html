Solving recurrences by induction

Example 1: Binary search

Let’s look at the following recurrence which describes the running time of the Binary Search algorithm:

T(n)=T(n/2)+1T(n)=T(n/2)+1

Additionally we are given the following information:

  • T(1)=1T(1)=1
  • nn is an exact power of two (i.e. n=2kn=2k)

We expand the equation of the recurrence to find a closed form:

T(n)=T(n/2)+1=(T(n/4)+1)+1=(T(n/8)+1)+1+1⋮=T(n/2k)+kT(n)=T(n/2)+1=(T(n/4)+1)+1=(T(n/8)+1)+1+1⋮=T(n/2k)+k

Now we will use induction to prove the correctness of this closed form:

  • as we know that n=2kn=2k, we can substitute into the closed form:

T(2k)=T(2k/2k)+kT(2k)=T(2k/2k)+k

Proposition: For any k∈N0k∈N0 the above equation holds.

Base case (Induktionsanfang):
Show that the proposition holds for k=0k=0:

T(20)=T(20/20)+0T(1)=1T(20)=T(20/20)+0T(1)=1

We can see that this is true given the information provided above.

Assumption (Induktionsbehauptung):
For a fixed k>0k>0 we have that T(2k)=T(2k/2k)+kT(2k)=T(2k/2k)+k

Inductive step (Induktionsschritt):
Show that the assumtion is true for k+1k+1:

T(2k+1)=T(2k+1/2k+1)+k+1T(2k+1)=T(2k+1/2k+1)+k+1

We can immediately see that this satisfies our assumption and therefore completes our proof.

If we resubstitute n=2kn=2k, we get:

T(n)=T(n/2k+1)+k+1T(n)=T(n/2k+1)+k+1

  • we know that n=2kn=2k
  • we also know that in each recursive step, the original problem size is reduced to a problem of size n/2kn/2k

→→ thus we know that the recursion will reach a boundary condition (i.e. T(1)T(1)), when 2k=n2k=n, or alternatively when k=log2nk=log2⁡n

So in our closed form we substitute k+1=log2nk+1=log2⁡n and get the following:

T(n)=T(n/2log2n)+log2n=T(1)+log2n=1+log2n=O(log2n)T(n)=T(n/2log2⁡n)+log2⁡n=T(1)+log2⁡n=1+log2⁡n=O(log2⁡n)

We have shown that the worst-case running time of Binary Search is T(n)=log2nT(n)=log2⁡n
