Solving recurrences using induction

Example 1:

Let’s look at the following recurrence which describes the running time of the Binary Search algorithm:

T(n)=T(n/2)+1T(n)=T(n/2)+1

Additionally we are given the following information:

  • T(1)=1T(1)=1
  • nn is an exact power of two (i.e. n=2kn=2k)

We expand the equation of the recurrence to find a closed form:

T(n)=T(n/2)+1=(T(n/4)+1)+1=(T(n/8)+1)+1+1⋮=T(n/2k)+kT(n)=T(n/2)+1=(T(n/4)+1)+1=(T(n/8)+1)+1+1⋮=T(n/2k)+k

Now we will use induction to prove the correctness of this closed form:

  • as we know that n=2kn=2k, we can substitute into the closed form:

T(2k)=T(2k/2k)+kT(2k)=T(2k/2k)+k

Proposition: For any k∈N0k∈N0 the above equation holds.

Base case (Induktionsanfang):
Show that the proposition holds for k=0k=0:

T(20)=T(20/20)+0⟺T(1)=1T(20)=T(20/20)+0⟺T(1)=1

We can see that this is true given the information provided above.

Assumption (Induktionsbehauptung):
For an arbitrary but fixed k>0k>0 we have that T(2k)=T(2k/2k)+kT(2k)=T(2k/2k)+k

Inductive step (Induktionsschritt):
Show that the assumtion is true for k+1k+1:

T(2k+1)=T(2k+1/2k+1)+k+1T(2k+1)=T(2k+1/2k+1)+k+1

We can immediately see that this satisfies our assumption and therefore completes our proof ■◼.

If we re-substitute n=2kn=2k, we get:

T(n)=T(n/2k+1)+k+1T(n)=T(n/2k+1)+k+1

  • we know that n=2kn=2k
  • we also know that in each recursive step, the original problem size is reduced to a problem of size n/2kn/2k

→→ thus we know that the recursion will reach a boundary condition (i.e. T(1)T(1)), when 2k=n2k=n, or alternatively when k=log2nk=log2⁡n

So in our closed form we substitute k+1=log2nk+1=log2⁡n and get the following:

T(n)=T(n/2log2n)+log2n=T(1)+log2n=1+log2n=O(log2n)T(n)=T(n/2log2⁡n)+log2⁡n=T(1)+log2⁡n=1+log2⁡n=O(log2⁡n)

We have shown that the worst-case running time of Binary Search is T(n)=O(log2n)T(n)=O(log2⁡n)

Example 2:

We are given the following recursion for the function TT:

T(n)=n⋅T(n2)T(n)=n⋅T(n2)

Additionally we are given the following information:

  • T(1)=1T(1)=1
  • n=2kn=2k, with k∈Nk∈N

We will use the given information and substitute as early as possible, which usually makes solving the recurrence easier:

T(2k)=2k⋅T(2k2)=2k⋅T(2k−1)T(2k)=2k⋅T(2k2)=2k⋅T(2k−1)

Next, we expand the equation of the recurrence to find a closed form:

T(2k)=2k⋅T(2k−1)=2k⋅2k−1⋅T(2k−2)=2k⋅2k−1⋅2k−2⋅T(2k−3)⋮T(2k)=2k⋅T(2k−1)=2k⋅2k−1⋅T(2k−2)=2k⋅2k−1⋅2k−2⋅T(2k−3)⋮

The recursion follows a very familiar pattern. It becomes clear that eventually we will meet a boundary condition (i.e. n=20=1n=20=1):

T(2k)=2k⋅2k−1⋅2k−2⋅...⋅21⋅ 20=2k+(k−1)+(k−2)+...+1T(2k)=2k⋅2k−1⋅2k−2⋅...⋅21⋅ 20=2k+(k−1)+(k−2)+...+1

Here we made use of the simple product rule for exponentiation with integers:

xm⋅xn=xm+nxm⋅xn=xm+n

We can express this summation like this:

=2∑ki=1i=2∑i=1ki

The summation in the exponent of the above term is an arithmetic series.
This arithmetic series has the closed form: ∑ni=1i=n(n+1)2∑i=1ni=n(n+1)2
Thus, we can re-write the above term like this (n=kn=k):

=2k(k+1)2=2k(k+1)2

So we have derived the closed form: T(2k)=2k(k+1)2T(2k)=2k(k+1)2

Again, we will use induction to prove the correctness of this closed form:

Proposition: For any k∈N0k∈N0 we have that T(2k)=2k(k+1)2T(2k)=2k(k+1)2

Base case (Induktionsanfang):
Show that the proposition holds for k=0k=0:

T(20)=20(0+1)2⟺T(1)=1T(20)=20(0+1)2⟺T(1)=1

Again, we can see that this is true given the information provided above.

Assumption (Induktionsbehauptung):
For an arbitrary but fixed k>0k>0 we have that T(2k)=2k(k+1)2T(2k)=2k(k+1)2

Inductive step (Induktionsschritt):
Show that the assumtion is true for k+1k+1:

To do this we substitute k=k+1k=k+1 in our original equation:

T(2k)=2k⋅T(2k−1)T(2k)=2k⋅T(2k−1)

T(2k+1)=2k+1⋅T(2(k+1)−1)=2k+1⋅T(2k)T(2k+1)=2k+1⋅T(2(k+1)−1)=2k+1⋅T(2k)

Holding on to our assumption that T(2k)=2k(k+1)2T(2k)=2k(k+1)2, for k>0k>0, we get:

=2k+1⋅2k(k+1)2=2k+1+k(k+1)2=2k+1⋅2k(k+1)2=2k+1+k(k+1)2

Here we again made use of the simple product rule for exponentiation with integers:

xm⋅xn=xm+nxm⋅xn=xm+n

=2k+1+k(k+1)2=22(k+1)2+k(k+1)2=22(k+1)+k(k+1)2=2k+1+k(k+1)2=22(k+1)2+k(k+1)2=22(k+1)+k(k+1)2

Let’s now quickly shift focus to the term in the exponent:

2(k+1)+k(k+1)2=(k+1)+(k+1)+k(k+1)2=(k+1)+(k+1)(k+1)2=(k+1)⋅((k+1)+1)22(k+1)+k(k+1)2=(k+1)+(k+1)+k(k+1)2=(k+1)+(k+1)(k+1)2=(k+1)⋅((k+1)+1)2

As you can see, we reached the final form by factoring out the term (k+1)(k+1) in the last step

When we substitute this term into our equation for T(2k+1T(2k+1) we end up with:

T(2k+1)=2(k+1)⋅((k+1)+1)2T(2k+1)=2(k+1)⋅((k+1)+1)2

From this form we can see that our assumption is satisfied and therefore our proof is complete ■◼.

Next, we want to re-substitute 2k=n2k=n into our original closed form: T(2k)=2k(k+1)2T(2k)=2k(k+1)2

  • we know that 2k=n2k=n, so we can derive that k=log2nk=log2⁡n by using the logarithm
  • →→ thus we can write 2log2n=n2log2⁡n=n

With this knowledge we can now get back to our original closed form:

T(2k)T(2log2n)T(n)=2k(k+1)2⟺=2log2n(log2n+1)2=2log2n⋅log2n+12⟺=nlog2n+12=n12⋅log2n+1T(2k)=2k(k+1)2⟺T(2log2⁡n)=2log2⁡n(log2⁡n+1)2=2log2⁡n⋅log2⁡n+12⟺T(n)=nlog2⁡n+12=n12⋅log2⁡n+1

We have now determined the following final closed form for the recursion:

T(n)=n12⋅log2n+1T(n)=n12⋅log2⁡n+1

Example 3:

We are given the following recursion for the function TT:

T(n)=2⋅T(23n)+1T(n)=2⋅T(23n)+1

Additionally we are given the following information:

  • T(1)=1T(1)=1
  • n=(32)kn=(32)k, with k∈Nk∈N

Again, we will substitute as early as possible to make solving the recurrence easier:

T(n)T((32)k)=2⋅T(23n)+1⟺=2⋅T(23⋅(32)k)+1=2⋅T((32)k/32)+1=2⋅T((32)k−1)+1T(n)=2⋅T(23n)+1⟺T((32)k)=2⋅T(23⋅(32)k)+1=2⋅T((32)k/32)+1=2⋅T((32)k−1)+1

To make the equation more readable, we substitute 32=x32=x and get:

T(xk)=2⋅T(xk−1)+1T(xk)=2⋅T(xk−1)+1

Next, we expand the equation of the recurrence to find a closed form:

T(xk)=2⋅T(xk−1)+1=2⋅(2⋅T(xk−2+1))+1=22⋅T(xk−2)+2+1=22⋅(2⋅T(xk−3)+1)+2+1=23⋅T(xk−3)+4+2+1=23⋅T(xk−3)+22+21+20⋮T(xk)=2⋅T(xk−1)+1=2⋅(2⋅T(xk−2+1))+1=22⋅T(xk−2)+2+1=22⋅(2⋅T(xk−3)+1)+2+1=23⋅T(xk−3)+4+2+1=23⋅T(xk−3)+22+21+20⋮

Once again, we can see that the recursion follows a clear pattern.

For any xkxk, where k∈Nk∈N, the recursion will at some point reach a boundary condition (i.e. T(xk−k)=T(x0)=T(1)=1T(xk−k)=T(x0)=T(1)=1), so we get:

T(xk)=2k+2k−1+2k−2+...+21+20T(xk)=2k+2k−1+2k−2+...+21+20

This summation can be expressed in the following way:

T(xk)=∑i=0k2iT(xk)=∑i=0k2i

The right term of the above equation is a geometric series.
The geometric series has the closed form: ∑ki=0ai=ak+1−1a−1∑i=0kai=ak+1−1a−1
Thus, we can re-write the above equation like this:

T(xk)=∑i=0k2i=2k+1−12−1=2k+1−1T(xk)=∑i=0k2i=2k+1−12−1=2k+1−1

Once again, we will use induction to prove the correctness of this closed form:

Proposition: For any k∈N0k∈N0 we have that T(xk)=2k+1−1T(xk)=2k+1−1

Base case (Induktionsanfang):
Show that the proposition holds for k=0k=0:

T(x0)=20+1−1⟺T(1)=1T(x0)=20+1−1⟺T(1)=1

Again, we can see that this is true given the information provided above.

Assumption (Induktionsbehauptung):
For an arbitrary but fixed k>0k>0 we have that T(xk)=2k+1−1T(xk)=2k+1−1

Inductive step (Induktionsschritt):
Show that the assumtion is true for k+1k+1:

To do this we substitute k=k+1k=k+1 in our original equation:

T(xk)=2⋅T(xk−1)+1T(xk)=2⋅T(xk−1)+1

T(xk+1)=2⋅T(x(k+1)−1)+1=2⋅T(xk)+1T(xk+1)=2⋅T(x(k+1)−1)+1=2⋅T(xk)+1

Holding on to our assumption that T(xk)=2k+1−1T(xk)=2k+1−1, for k>0k>0, we get:

=2⋅T(xk)+1=2⋅(2k+1−1)+1=21⋅2k+1−21+1=2k+2−2+1=2k+2−1=2⋅T(xk)+1=2⋅(2k+1−1)+1=21⋅2k+1−21+1=2k+2−2+1=2k+2−1

From this form we can see that our assumption is satisfied and therefore our proof is complete ■◼.

Next, we want to re-substitute x=32x=32 and n=(32)kn=(32)k into our original closed form: T(xk)=2k+1−1T(xk)=2k+1−1

(x=32x=32):

T(xk)T((32)k)=2k+1−1⟺=2k+1−1T(xk)=2k+1−1⟺T((32)k)=2k+1−1

  • we know that (32)k=n(32)k=n, so we can derive that k=log3/2nk=log3/2⁡n by using the logarithm
  • →→ thus we can write (32)log3/2n=n(32)log3/2⁡n=n

With this knowledge we can now get back to our original closed form:

(n=(32)kn=(32)k):

T((32)k)T((32)log3/2n)T(n)=2k+1−1⟺=2log3/2(n)+1−1⟺=2log3/2(n)+1−1T((32)k)=2k+1−1⟺T((32)log3/2⁡n)=2log3/2⁡(n)+1−1⟺T(n)=2log3/2⁡(n)+1−1

We have now determined the following final closed form for the recursion:

T(n)=2log3/2(n)+1−1T(n)=2log3/2⁡(n)+1−1

Example 4:

We are given the following recursion for the function TT:

T(n)=n2⋅T(n/2)T(n)=n2⋅T(n/2)

Additionally we are given the following information:

  • T(1)=1T(1)=1
  • n=2kn=2k, with k∈Nk∈N

Like in the examples above, we will first substitute n=2kn=2k:

T(n)T(2k)=n2⋅T(n/2)⟺=2k2⋅T(2k/2)=2k−1⋅T(2k−1)T(n)=n2⋅T(n/2)⟺T(2k)=2k2⋅T(2k/2)=2k−1⋅T(2k−1)

Next, we expand the equation of the recurrence to find a closed form:

T(2k)=2k−1⋅T(2k−1)=2k−1⋅(2k−2⋅T(2k−2))=2k−1⋅2k−2⋅(2k−3⋅T(2k−3))⋮T(2k)=2k−1⋅T(2k−1)=2k−1⋅(2k−2⋅T(2k−2))=2k−1⋅2k−2⋅(2k−3⋅T(2k−3))⋮

From the pattern of the recursion, we can see that for any k∈Nk∈N we will at some point reach a boundary condition (i.e. T(2k−k)=T(20)=T(1)=1T(2k−k)=T(20)=T(1)=1).
Therefore, we get:

T(2k)=2k−1⋅2k−2⋅2k−3⋅...⋅21⋅20=2(k−1)+(k−2)+(k−3)+...+1T(2k)=2k−1⋅2k−2⋅2k−3⋅...⋅21⋅20=2(k−1)+(k−2)+(k−3)+...+1

Here we again made use of the simple product rule for exponentiation with integers:

xm⋅xn=xm+nxm⋅xn=xm+n

Let’s have a closer look at the summation that is happening in the exponent of the above term:

(k−1)+(k−2)+(k−3)+...+1(k−1)+(k−2)+(k−3)+...+1

We can express the summation like this:

=∑i=1k−1i=∑i=1k−1i

The above term is an arithmetic series.
This arithmetic series has the closed form: ∑ni=1i=n(n+1)2∑i=1ni=n(n+1)2
Thus, we can re-write the above term like this:
(n=(k−1)n=(k−1))

=(k−1)((k−1)+1)2=(k−1)((k−1)+1)2

Going back to our complete equation, thus get:

T(2k)T(2k)T(2k)=2(k−1)+(k−2)+(k−3)+...+1⟺=2∑k−1i=0i⟺=2(k−1)((k−1)+1)2T(2k)=2(k−1)+(k−2)+(k−3)+...+1⟺T(2k)=2∑i=0k−1i⟺T(2k)=2(k−1)((k−1)+1)2

We now have derived the following closed form for our recurrence:

T(2k)=2(k−1)((k−1)+1)2T(2k)=2(k−1)((k−1)+1)2

Once again, we will use induction to prove the correctness of this closed form:

Proposition: For any k∈N0k∈N0 we have that T(2k)=2(k−1)((k−1)+1)2T(2k)=2(k−1)((k−1)+1)2

Base case (Induktionsanfang):
Show that the proposition holds for k=0k=0:

T(20)T(1)=2(0−1)((0−1)+1)2=2−1(−1+1)2=2−1⋅02=202=20⟺=1T(20)=2(0−1)((0−1)+1)2=2−1(−1+1)2=2−1⋅02=202=20⟺T(1)=1

Again, we can see that this is true given the information provided above.

Assumption (Induktionsbehauptung):
For an arbitrary but fixed k>0k>0 we have that T(2k)=2(k−1)((k−1)+1)2T(2k)=2(k−1)((k−1)+1)2

Inductive step (Induktionsschritt):
Show that the assumtion is true for k+1k+1:

To do this we substitute k=k+1k=k+1 in our original closed form:

T(2k)=2(k−1)((k−1)+1)2T(2k)=2(k−1)((k−1)+1)2

T(2k+1)=2((k+1)−1)[((k+1)−1)+1]2=2k(k+1)2T(2k+1)=2((k+1)−1)[((k+1)−1)+1]2=2k(k+1)2

From this form we can see that our assumption is satisfied and therefore our proof is complete ■◼.

Next, we want to re-substitute n=2kn=2k into our original closed form: T(2k)=2(k−1)((k−1)+1)2T(2k)=2(k−1)((k−1)+1)2

  • we know that 2k=n2k=n, so we can derive that k=log2nk=log2⁡n by using the logarithm
  • →→ thus we can write n=2log2nn=2log2⁡n

We can now return to our original closed form:

T(2k)T(2log2n)T(n)=2(k−1)((k−1)+1)2⟺=2((log2n)−1)(((log2n)−1)+1)2=2((log2n)−1)(log2n)2=2log2n⋅((log2n)−1)2⟺=n12⋅(log2n)−1T(2k)=2(k−1)((k−1)+1)2⟺T(2log2⁡n)=2((log2⁡n)−1)(((log2⁡n)−1)+1)2=2((log2⁡n)−1)(log2⁡n)2=2log2⁡n⋅((log2⁡n)−1)2⟺T(n)=n12⋅(log2⁡n)−1

We have now determined the following final closed form for the recursion:

T(n)=n12⋅(log2n)−1T(n)=n12⋅(log2⁡n)−1
