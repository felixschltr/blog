5. Quicksort

Quicksort is a divide-and-conquer algorithm for sorting in place and uses the following three steps:

Divide:

  • the original input array A[left...right]A[left...right] is partitioned into two subarrys A[left...pivot−1]A[left...pivot−1] and A[pivot+1...right]A[pivot+1...right], such that:

∀u∈A[left...pivot−1]≤v∈A[pivot]≤∀w∈A[pivot+1...right]∀u∈A[left...pivot−1]≤v∈A[pivot]≤∀w∈A[pivot+1...right]

Conquer:

  • the two subarrays A[left...pivot−1]A[left...pivot−1] and A[pivot+1...right]A[pivot+1...right] are sorted by recursive calling quicksort

Combine:

  • the combine step consumes no time, as the two subarrays are already sorted

The ‘Partition’ subprocedure

In the partition() subprocedure, an element in the array is chosen as ‘pivot element’, which will serve as a reference for partitioning the original input array into two subarrays (as described in the combine step above).





















public int partition(int[] arr, int left, int right) {
    //declare pivot element to be last element in array
    int pivot = arr[right];
    int i = left - 1;
    for(int j = left; j <= right -1; j++ ) {
        if(arr[j] <= pivot) {
            i ++;
            // swap elements arr[i] & arr[j]
            int temp = arr[i];
            arr[i] = arr[j];
            arr[j] = temp;
        }
    }
    //swap elements arr[i+1] & arr[right](= pivot element)
    int temp = arr[i+1];
    arr[i+1] = arr[right];
    arr[right] = temp;
    //return the new index of pivot element
    return i+1;
}

The Quicksort algorithm

The main algorithm of Quicksort uses recursive calls to the partition() subprocedure to sort the original input array in place.











public void quickSort(int[] arr, int left, int right) {
    if(left < right) {
        //determine index of pivot element
        int pivot = partition(arr,left,right);
        //recursive calls to quickSort()
        quickSort(arr, left, pivot-1);
        quickSort(arr, pivot + 1, right);

    }
}
